# VA皮肤科小红书自动运营 Skill

## 配置向导

**触发条件**：用户提到"配置向导"、"初始化"、"设置诊所"、"配置诊所"、"重新配置"，或首次使用时 `config/clinic.json` 不存在/为空。

**向导流程**：

1. 检查 `config/clinic.json` 是否存在且四个字段均非空
2. 如果未配置，依次询问用户：
   - **诊所名称**（`clinic`）：评论中出现的品牌简称，如 `VA`、`曙光`、`XX皮肤科`
   - **所在地区**（`location`）：诊所所在的标志性区域，如 `梨大`、`江南`、`新沙`
   - **医生职称**（`title`）：主理医生的职称，如 `大教授`、`主任医师`、`教授`
   - **负责人称谓**（`role`）：负责人的称谓，如 `园长`、`院长`、`主任`
3. 将结果写入 `config/clinic.json`
4. 回复："✅ 配置完成，开始为 **{诊所名称}**（{地区}）运营"

> 重置配置：删除 `config/clinic.json` 后重新运行脚本即可。
